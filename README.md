**E-Commerce Web Application**

This is a simple E-Commerce web application built using Django REST Framework and React.
The project focuses on backend APIs, token-based authentication, and frontendâ€“backend integration.

**Features**

User registration and login

JWT token-based authentication

Product listing from backend APIs

Add products to cart

View and manage cart

Place order from cart

**Tech Stack**

**Backend**

Python

Django

Django REST Framework

JWT Authentication

**Frontend**

React

Axios

HTML, CSS, JavaScript

**How it works**

User logs in and gets a JWT token

Token is used to access protected APIs

Frontend communicates with backend using REST APIs

Cart and order logic handled in backend

**Project Status**

Core functionality is implemented.
Some UI improvements and edge-case handling are in progress.

**How to run**

**Backend**

python manage.py runserver


**Frontend**

npm start
